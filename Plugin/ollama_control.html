Ollama画板控制

注意： 此界面需要您手动启动Python插件。请在终端中运行： python ollama_draw_plugin.py

连接设置

请检查Ollama和画板服务是否正在运行

对话控制

请输入自然语言指令控制画板（例如："在画布中间画一个红色圆形"）
可用命令示例：
  • 绘制基本图形："在画布中间画一个黑色圆形"、"从左上角到右下角画一条红色直线"
  • 设置样式："使用绿色绘制"、"将线宽设置为5"
  • 文本操作："在画布底部写'Hello World'"
  • 其他操作："清空画布"、"画一个蓝色正方形"
